# Handshake Trust Checker:

A Chrome extension that automatically analyzes job postings on Handshake and flags potential scams or ghost jobs to protect students during their job search.

## What It Does:

Handshake Trust Checker scans job listings in real-time and identifies red flags commonly associated with:
Scam Jobs - Fake postings designed to steal money or personal information
Ghost Jobs - Positions that companies post but never intend to fill
Suspicious Postings - Jobs with unprofessional or misleading content

  <br/>

## Installation:

Works with Chrome and Edge browsers. 

1. Download the Extension

   - Click the green "Code" button on this GitHub page

   - Select "Download ZIP"

2. Extract the ZIP file to a folder on your computer

3. Open Chrome or Edge

4. Navigate to chrome://extensions/ (or edge://extensions/), or select the extensions icon from the top right (puzzle piece) <br/>

   <img width="298" height="97" alt="image" src="https://github.com/user-attachments/assets/10020717-1bf9-4ed3-b23a-897d5bb4e0f1" />


5. Enable "Developer mode" in the top-right corner


6. Click "Load unpacked" <br/>

   <img width="418" height="414" alt="image" src="https://github.com/user-attachments/assets/5aef1fb6-cf83-49b0-ab6d-719dca5c18ce" />
